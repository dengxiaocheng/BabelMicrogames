# BabelMicrogames

This umbrella repository is not used for game source code.

Microgames are split into one repository per game:

- `BabelMicrogame-PeigeiRi`
- `BabelMicrogame-YejianQidao`
- `BabelMicrogame-GongtouDianming`

Each JavaScript microgame repository owns its own source, issue queue,
ClaudeCode workdir, and GitHub Pages link.
