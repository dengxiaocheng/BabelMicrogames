# Incoming Design Drop Zone

Place newly added design documents here first.

Typical contents:

- revised system design notes
- Babel/C++ integration notes
- Godot-facing gameplay requirements
- updated gameplay validation summaries
- requirement-system proposals

This directory is a staging area.

After review, important content should be:

- merged into canonical docs under `docs/`
- or archived into `docs/planning/`

Do not treat this directory as the final documentation source of truth.
